The MongoDB component is a data component that allows users to access the MongoDB NoSQL database.
It can carry out the following tasks:

- [Insert](#insert)
- [Find](#find)
- [Update](#update)
- [Delete](#delete)
- [Drop Collection](#drop-collection)
- [Drop Database](#drop-database)



## Release Stage

`Alpha`



## Configuration

The component configuration is defined and maintained [here](https://github.com/instill-ai/component/blob/main/data/mongodb/v0/config/definition.json).




## Setup


| Field | Field ID | Type | Note |
| :--- | :--- | :--- | :--- |
| URI (required) | `uri` | string | Fill in your MongoDB URI |




## Supported Tasks

### Insert

Perform an insert operation


| Input | ID | Type | Description |
| :--- | :--- | :--- | :--- |
| Task ID (required) | `task` | string | `TASK_INSERT` |
| Database Name (required) | `database-name` | string | Fill in your MongoDB database name |
| Collection Name (required) | `collection-name` | string | The name of the collection in the MongoDB database |
| Data (required) | `data` | semi-structured/json | The data to be inserted |



| Output | ID | Type | Description |
| :--- | :--- | :--- | :--- |
| Status | `status` | string | Insert status |






### Find

Perform a find operation


| Input | ID | Type | Description |
| :--- | :--- | :--- | :--- |
| Task ID (required) | `task` | string | `TASK_FIND` |
| Database Name (required) | `database-name` | string | Fill in your MongoDB database name |
| Collection Name (required) | `collection-name` | string | The name of the collection in the MongoDB database |
| Criteria (required) | `criteria` | semi-structured/json | The criteria to find documents, if JSON value of a key is null e.g \{'name':null\}, then name field will be selected without any criteria, if JSON is empty e.g \{\}, then all fields will be selected |
| Limit (required) | `limit` | integer | The number of documents to return, if 0 then all documents will be returned, otherwise the number of documents to return |



| Output | ID | Type | Description |
| :--- | :--- | :--- | :--- |
| Documents | `documents` | array | The documents returned from the find operation |
| Status | `status` | string | Find status |
